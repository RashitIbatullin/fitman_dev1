# Фитнес-менеджер (FitMan) - DEV1

Система автоматизации процессов фитнес-центра.

## 🚀 Возможности

- Управление помещениями
- Управление оборудованием
- Управление пользователями (Клиенты, Инструкторы, Тренеры, Администраторы)
- Планы тренировок
- Контроль антропометрии
- Расписание занятий
- Интеграция с фин.системами

## 🛠 Технологии

### Backend
- Dart Frog (Dart)
- Shelf (Server) 
- PostgreSQL
- JWT аутентификация

### Frontend
- Flutter (Dart)
- Riverpod для управления состоянием
- Go Router для навигации

## 📦 Установка

### Предварительные требования
- Dart SDK 3.0+
- Flutter SDK 3.0+
- PostgreSQL 15+
